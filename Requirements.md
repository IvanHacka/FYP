User Requirements

## Authentication and Access Control

- As a user, I want to register an account so that I can access the job board system.
- As a user, I want to log in securely so that I can access my personalised features.
- As a user, I want to log out so that my account remains secure on shared devices.
- As a developer, I want role-based access control so that employees, employers, and admins can only access authorised features.

------------------------------------------------------------------

## Employee Requirements

### Profile Management
- As an employee, I want to create and update my profile so that employers can view my information.
- As an employee, I want to add personal details such as name, email, phone number, and bio so that my profile is complete.
- As an employee, I want to upload my CV so that I can apply for jobs.
- As an employee, I want to upload other supporting documents so that employers can review more of my information.
- As an employee, I want to update or replace my uploaded documents so that my records stay current.

### Job Browsing and Filter Searching
- As an employee, I want to browse available jobs so that I can find suitable opportunities.
- As an employee, I want to search jobs by title so that I can find relevant roles more quickly.
- As an employee, I want to search jobs by location so that I can find jobs in suitable areas.
- As an employee, I want to filter jobs by salary so that I can focus on roles that meet my expectations.
- As an employee, I want to view detailed job information so that I can decide whether to apply.

### Watch List
- As an employee, I want to save jobs to a watch list so that I can return to them later.
- As an employee, I want to remove jobs from my watch list so that I can keep it organised.
- As an employee, I want to view all saved jobs in one place so that I can manage my interests easily.

### Job Application
- As an employee, I want to apply for a job so that I can be considered for the role.
- As an employee, I want to provide job-specific information during the application process so that each application is tailored to the role.
- As an employee, I want to enter my expected salary when applying so that employers can assess my expectations.
- As an employee, I want to explain why I am a good fit so that I can strengthen my application.
- As an employee, I want to provide my available start date so that employers can consider my availability.
- As an employee, I want the system to prevent duplicate applications so that I do not apply to the same job twice by mistake.
- As an employee, I want validation on application inputs so that I do not submit invalid information.

### Application Tracking
- As an employee, I want to view all my submitted applications so that I can track my progress.
- As an employee, I want to see the current status of each application so that I know whether it is under review, shortlisted, rejected, or otherwise updated.
- As an employee, I want to withdraw an application so that I can cancel it when necessary.
- As an employee, I want to view my application timeline so that I can understand the progress of my application.

## Employer Requirements

### Employer Profile Management
- As an employer, I want to create and update my company profile so that applicants can view my company information.
- As an employer, I want to add company name, website, and description so that my organisation is represented clearly.

### Job Management
- As an employer, I want to post a new job so that I can attract candidates.
- As an employer, I want to edit my job postings so that I can correct or improve job information.
- As an employer, I want to delete a job posting so that I can remove roles that are no longer needed.
- As an employer, I want to close a job posting so that applications are no longer accepted.
- As an employer, I want to set a job expiry date so that jobs are automatically limited to a relevant period.
- As an employer, I want to view the status of my jobs so that I know whether they are open, closed, or expired.

### Job Skills Management
- As an employer, I want to add required skills to a job so that I can define job expectations clearly.
- As an employer, I want to update the importance level of required skills so that matching can better reflect the needs of the job.
- As an employer, I want to remove job skills so that I can keep the job requirements accurate.

### Applicant Review
- As an employer, I want to view all applicants for a job so that I can assess candidates.
- As an employer, I want to view applicant profile details so that I can better understand their background.
- As an employer, I want to view applicant CVs so that I can assess qualifications.
- As an employer, I want to view all applicant uploaded documents so that I can review supporting materials beyond the CV.
- As an employer, I want to download applicant documents so that I can store or review them offline.
- As an employer, I want to see a match score for each applicant so that I can quickly compare suitability.
- As an employer, I want to see a score breakdown by skills, title, location, salary, and job type so that I can understand how the match score was calculated.

### Application Management
- As an employer, I want to update application statuses so that I can manage candidates through the recruitment process.
- As an employer, I want to mark applications as under review so that applicants know they are being considered.
- As an employer, I want to shortlist applicants so that I can identify stronger candidates.
- As an employer, I want to reject applicants so that unsuitable candidates can be filtered out.
- As an employer, I want to manage later recruitment stages such as interview and offer so that the recruitment workflow is more complete.
- As an employer, I want to add internal notes to applications so that I can record recruitment decisions and comments.

------------------------------------------------------------------

## Admin Requirements

- As an admin, I want to manage user access so that the system remains secure and controlled.
- As an admin, I want to oversee employee and employer accounts so that inappropriate or invalid accounts can be handled.
- As an admin, I want the system to enforce permissions so that users cannot access features outside their role.
